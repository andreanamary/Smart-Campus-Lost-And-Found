# Smart Campus Lost & Found Management System

## Description

A Flask-based web application developed for managing lost and found items inside a college campus.

## Features

- Student Registration
- Student Login
- Dashboard
- Dashboard Statistics
- Report Lost Item
- Report Found Item
- Search Lost Items
- SQLite Database
- Responsive Interface

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
